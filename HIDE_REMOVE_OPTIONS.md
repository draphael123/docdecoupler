## ðŸ”§ Technical Details

### **Filtering Logic**

The system applies filters in this order:

1. **Quick Filters** (onlyShowUnique/onlyShowShared)
   - Overrides include/exclude settings
   
2. **Match Type Filters**
   - hideExactMatches: Excludes matchType === 'exact'
   - hideFuzzyMatches: Excludes matchType === 'fuzzy'
   - hideUserOverrides: Excludes matches with userOverride set

3. **Confidence Filter**
   - hideLowConfidence: Excludes matches below threshold
   - confidenceThreshold: Minimum confidence (0-1)

4. **Formatting Filters**
   - hideSectionHeaders: Removes section titles
   - hideFooters: Removes page footers
   - hideSourceInfo: Removes metadata section

### **Statistics Calculation**

The preview stats automatically update to reflect:
- Filtered match counts
- Excluded content
- Final line counts
